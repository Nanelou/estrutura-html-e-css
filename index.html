Função do HTML e CSS na Programação

HTML (HyperText Markup Language)

O HTML é a base estrutural de qualquer página web...

Explicando o código:

  • <!DOCTYPE html>: Este comando informa ao navegador que o arquivo é um documento HTML5.
  • <html lang="pt-br">: Define que o conteúdo da página é em português do Brasil.
  • <head>: A parte do código onde colocamos informações importantes sobre a página, como o título da página e links para o CSS.
  • <meta charset="UTF-8">: Diz ao navegador que os caracteres usados no código são em UTF-8, o que permite mostrar letras e símbolos especiais corretamente (como acentos e cedilhas).
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Faz com que a página seja responsiva, ou seja, se ajuste ao tamanho da tela do dispositivo (como celular ou computador).
  • <title>: Define o título que aparece na aba do navegador.
  • <link rel="stylesheet" href="styles.css">: Este comando conecta o arquivo CSS com o arquivo HTML. Assim, o estilo da página será definido pelo código CSS que criamos.
  • <body>: Dentro dessa tag está o conteúdo visível da página. Tudo o que você vê no site estará dentro do <body>.
  • <h1>, <h2>, <p>: Estas são tags de cabeçalhos e parágrafos que ajudam a organizar o texto e estruturar a página de forma hierárquica.

CSS (Cascading Style Sheets)

O CSS é utilizado para estilizar o HTML, definindo cores, tamanhos, fontes e posicionamento dos elementos....

Conclusão

O HTML define o que será exibido, enquanto o CSS define como será exibido...
